#列表
	变量名称 = [元素1， 元素2， 元素3...]
	空列表：变量名称 = []
		变量名称 = list()
	下标索引：0  -  n-1或-1  -  -n
列表方法：
	1、查询：列表.index(元素)，查找成功，返回正索引值；失败，直接报错
	2、修改：列表[元素] = 值
	3、插入：列表.insert(下标，元素)
	4、追加：列表.append(元素)或列表.extend(其他数据容器)
	5、删除：del 列表[index]或列表.pop(index)，pop返回删除元素
		列表.remove(元素)删除第一个匹配项
	6、清空：列表.clear()
	7、统计：列表.count(元素)统计某元素个数
		len(列表)统计列表元素个数

#元组(不可修改)
	变量名称 = (元素1， 元素2， ......)
	空元组：变量名称 = ()
		变量名称 = tuple()
	单元素：(元素，)得加逗号
	下标索引：0-n-1
列表方法：
	1、index()
	2、count()
	3、len(元组)
但是元组中的list可以修改

#字符串也是容器(无法修改字符串内容)
	1、字符串替换：字符串.replace(字符串1，字符串2)将字符串内所有的字符串1替换为2
	不是修改，而是新的副本
	2、字符串分割：字符串.split(分割字符串)按照指定的分割串将字符串分为多个串，放入列表中
	3、字符串规整：字符串.strip()（去前后空格）
			字符串.strip(字符串)（删去指定字符串）
	4、统计子串出现次数count()
	5、计算串长len()

#集合
	变量名称 = {元素，... , 元素}
	空集合：变量名称 = set()
集合方法：
	添加：集合.add()
	移除：集合.remove()
	随机取出：集合.pop()
	清空：集合.clear()
	取差集：集合1.diffeence(集合2)	(集合1有而集合2没有的)
	消除差集：集合1.difference_update(集合2)	(集合1被修改，集合2不变)
	集合合并：集合1.union(集合2)
	统计元素个数：len()
	
#字典
	变量名称 = {key = value, key = value, . . .}
	空字典：变量名称 = {} 或 变量名称 = dict()
	key不能重复，不能为字典
字典方法：
	添加：字典[key] = value
	更新：字典[key] = value（对于已存在的，更新值）
	删除：字典.pop(key)，返回value
	清空：字典.clear()
	获取全部的key：字典.keys()
	统计数量：len()
	
五类容器可以互相转换
容器排序：sorted(容器，[reverse=True])，排完后都变成列表对象

#文件操作
	打开文件：open(name, mode, encoding)
		encoding直接指定，不能直接传参
	读取文件：f.read()参数可以指定读取字符数，默认全读取
		f.readlines()依次读一行，全部读取完存在列表中
		f.readline()一次读取一行
		for line in f:
			print(line)

	自动关闭文件：with open(name, mode, encoding) as 对象:
			operator
	写入文件：f.write(内容)	f.fllush()
		f.close()自动执行flush功能
