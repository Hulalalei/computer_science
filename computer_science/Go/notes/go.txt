1、初始化1：
    var a int 所有类型变量如果未赋值，则均为0
2、初始化2：
    a := 10 || var a = 10，也可以根据值来判断变量类型 

a := 10 等价于:
    var a int
    a = 10

= 表示赋值，:= 表示初始化声明，声明过后再:=会报错，只能在函数内部使用，且不能声明多次
局部变量必须使用，全局没有要求
向量式赋值：
    支持向量式赋值，逗号分隔
    并行接受多个参数：val, err = Func(var1)
    空白标识符：_, 表示只写，无法读取

常量：只能是bool、数字、字符串类型
iota特殊常量：计数const

运算符与cpp完全相同

if函数：
if condition {

} else {

}
条件函数与cpp类似，只不过函数前不加括号
数组大小不同，类型即不同
