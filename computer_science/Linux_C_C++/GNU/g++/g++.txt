GCC常见组成部分：c++、configure、g++、gcc、libgcc、libstdc++
GCC常见软件：ar、as、gdb、gprof、ld、libtool、make

源文件(hello.c) -> 预处理器：cpp -> 修改后源文件(hello.i) -> 编译器：ccl
-> 汇编程序(hello.s) -> 汇编器：as -> 可重定位目标程序(hello.o + lib.o)
-> 链接器：ld -> 可执行程序(hello.exe)

预处理：gcc -E main.c -o main.i
编译：gcc -S main.c -o main.s (得到汇编)
汇编：gcc -c main.c -o main.o

静态链接库(.a)：
1、编译出.o文件
2、ar -r [lib自定义库名.a] [.o] [.o]
3、链接：gcc [.c] [.a] -o name、gcc [.c] -o name -l lib -L libpath

动态链接库(.so)
1、编译出.o文件：gcc -c -fpic [.cc]
2、gcc -shared [.o] -o lib.so
3、链接：gcc [.cc] -o name -l lib -L libpath -Wl, -rpath=libpath

高版本gcc下载：
	yum install centos-release-scl
	yum install devtoolset-9
	scl enable devtoolset-9 bash
	root用户下两个配置文件~/.bashrc、~/.bash_profile均加上：
		if [ -f /opt/rh/devtoolset-9/enable ]; then
			. /opt/rh/devtoolset-9/enable
		fi
		export CC=gcc-9.3.1/10.2.1
		export CXX=g++-9.3.1/10.2.1

更换yum源：
	安装wget&epel第三方源
		yum install -y wget
		yum install epel-release
	备份
		cd /etc/yum.repos.d/
		mv CentOS-Base.repo CentOS-Base.repo.bak
	下载阿里云的CentOS-7.repo文件
		wget -O /etc/yum.repos.d/CentOS-Base.repo http://mirrors.aliyun.com/repo/Centos-7.repo
	重新加载yum
		yum clean all
		yum makecache fast
