=== 资本主义 ===
    == 帽子
        = 私人劳动和社会劳动的矛盾构成 《私有制基础上商品经济》的基本矛盾
        = 资本积累是资本主义社会失业现象产生的根源

    == 计算
        = 生产价格 = 生产成本 + 平均利润
        = 垄断价格 = 成本价格 + 平均利润 + 垄断利润
        = 剩余价值率 = 剩余价值(m) / 可变资本(v)
        = 利润 = 剩余价值m

    == 必要劳动 & 剩余劳动
        = 必要劳动：再生产劳动力的价值
        = 剩余劳动：无偿为资本家生产剩余价值

    == 不变资本(c) & 可变资本(v)
        = 不变资本：以生产资料形态存在的资本；通过工人的具体劳动被转移到新产品中
        = 可变资本：购买劳动力的资本，由工人的劳动再生产出来

    == 资本有机构成
        = 技术构成：由生产的技术水平所决定的生产资料和劳动力之间的比例
        = 价值构成：c : v
        = 有机构成：由技术构成引起的价值构成变化，c : v来表示

    == 货币资本 & 生产资本 & 商品资本
        = 在资本循环中执行的不同职能

    == 固定资本 & 流动资本
        = 固定资本：多次转移(盆)
        = 流动资本：一次转移(面)

                                                        思道法
=== 帽子
    = 人生目的 是 人生观的核心
    = 理想的特征：超越性、实践性、时代性
    = 信念的特征：执着性、支撑性、多样性

    = 以爱国主义 为核心的伟大民族精神
    = 以改革创新 为核心的时代精神

    = 社会主义核心价值观 要求：把涉及国家、社会、公民的价值要求融为一体；而社会主义核心价值体系没有这个要求
    = 社会主义道德：以人民服务为核心、以集体主义为原则

    = *1 法律的运行
        = 法律制定：是法律运行的起始性和关键性环节
        = 法律执行：是法律实施和实现的重要环节
        = 法律适用：司法；国家审判机关、检察机关
        = 法律遵守：是法律实施和实现的基本途径

    = *2 建设中国特色社会主义法治体系
        = 完备的法律规范体系，是法治体系的前提，是国家、政府、社会的制度基础
        = 高效的法治实施体系，是建设法治体系的重点
        = 严密的法治监督体系，是宪法法律有效实施的重要保障
        = 有力的法治保障体系，是全面依法治国的重要依托
        = 完善的党内法规体系，是法治体系的本质要求和重要内容

    = *3 坚持全面推进科学立法、严格执法、公正司法、全民守法
        = 科学立法是全面依法治国的前提
        = 严格执法是全面依法治国的关键
        = 公正司法是全面依法治国的重点
        = 全民守法是全面依法治国的基础

    = *4 权利保障
        = 宪法保障 是权利保障的前提和基础
        = 立法保障 是权利保障的重要条件
        = 行政保障 是权利保障的关键环节
        = 司法保障 是权利保障的的最后防线
